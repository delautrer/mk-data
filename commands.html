/start

Starts a new race series between teams.

Options:

  • teamtags (required) → Enter team tags for all competing teams in this series, separated by spaces.
  • racecount (optional) → Sets the number of races (default: 12).

Usage Examples:

/start $p t2

➝ Starts a new 6v6 race series between $p and t2 (with 12 races).

/start $p t2 t3

➝ Starts a new 4v4v4 race series between $p, t2, and t3.

Details:

  • Supported Formats: 6v6 (2 teams), 4v4v4 (3 teams), 3v3v3v3 (4 teams), 2v2v2v2v2v2 (6 teams) (depends on the number of unique team tags entered).
  • If a race series is already active, the new one will override the current one.
  • Once all races are completed, the race is moved to the history.
  • If not all races have been completed, the race series will be deleted.
  • The overlay will be automatically updated: If you enter an MKC team tag, the overlay will fetch the team's color and logo.

/back

Backs the last action for the current or last race. This includes only placing and penalties.

/end

If there is a race in the channel, it will be ended.

Details:

  • Once all races are completed, the race is moved to the history.
  • If not all races have been completed, the race series will be deleted.
  • Cannot be undone with /back.
  • The overlay will be automatically updated.

/penalty

Add a penalty to a given team tag.

Options:

  • score (required) → Adds the given score as a penalty.
  • racecount (optional) → Adds a penalty to the specified team tag. (Defaults to the first team tag entered with the /start command).

Usage Examples:

/penalty -20 t2

➝ Adds the penalty -20 to the score of t2.

/penalty -20

➝ Adds the penalty -20 to the score of $p.

Details:

  • The bot provides response options for the team tags.
  • Consecutive penalties are summed up.
  • The overlay will be automatically updated.

/edit

Allows you to customize team tags, add an MK-Central Id to a team tag, change track selection, and placements for a specific race.

Subcommands:

/edit teamtag

Manage the team tags of the current race.

  • /edit teamtag set <old> <new> → Changes a team tag from old to new.
  • /edit teamtag addid <teamtag> <mkcid> → Adds an MK-Central Id (mkcid) to the given team tag.

/edit race

Modify the data of the current race.

  • /edit race setranks <racenum> <placements> → Changes the placements for the given race (racenum).
  • /edit race settrack <racenum> <track abbr.> → Changes the track for the given race (racenum).

Usage Examples:

/edit teamtag t2 kw3

➝ Changes the team tag of t2 from t2 to kw3.

/edit teamtag addid kw3 2345

➝ Adds the MK-Central Id 2345 to kw3.

/edit race setranks 1 13-59

➝ Sets the placements of the 1st race to 13-59 (So: 1, 3, 4, 5, 9 & 12).

/edit race settrack 2 dbp

➝ Sets the track of the 2nd race to dbp (Baby Park).

Details:

  • The bot provides response options for the team tags and race.
  • /edit teamtag set ... → Only changes the tag if the team tags are valid and unique.
  • /edit teamtag addid ... → Only adds the ID to the team tag if the ID is valid.
  • /edit race setranks ... → Only changes the placements if the placement string and race number are valid.
  • /edit race settrack ... → Only changes the track if the track abbreviation and race number are valid.
